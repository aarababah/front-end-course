# Warm up

## 1.

Given this array:

```js
var array = [[[[[[[function () {
  console.log('Found it!');
}]]]]]];
```

Please call the anonymous function.

## 2.

Given this object:

```js
var object = {
  list: [[[[[[[{
    message: 'Found it!'
  }]]]]]]]
};
```

Please print the message.
